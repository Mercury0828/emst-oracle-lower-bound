# emst-oracle-lower-bound

Research project for idea `emst-oracle-lower-bound` (target venue: **SODA**).

A query lower bound: prove Ω(n^{1/2−o(1)}) orthogonal-range-counting-oracle queries are necessary to
(1±ε)-estimate the Euclidean MST weight, closing the Ω(n^{1/3}) vs Õ(√n) gap of Driemel et al.
(SoCG 2025, arXiv:2504.15292).

## How to run

Open this directory in Claude Code and paste the contents of `start_up_for_claude_code.md` into the
session — it bootstraps Phase 0. `guide.md` is the read-only project constitution (the full research
plan: model, theory program, the external-solver attack & audit loop, kill criteria, paper plan).
